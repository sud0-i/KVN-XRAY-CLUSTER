# KVN-XRAY-CLUSTER

Централизованная система для развертывания и управления кластером VPN-узлов на базе Xray-core. Проект использует архитектуру Master-Agent, написан на Go и управляется через единый Bash-интерфейс или Telegram-бота.

## Архитектура

* **Master API (`vpn-master`)**: Центральный узел. Хранит базу данных (SQLite), выдает конфигурации Агентам по защищенному токену, собирает статистику и обслуживает Telegram-бота.
* **Agent (`vpn-agent`)**: Устанавливается на конечные узлы (RU-мосты и EU-ноды). Автоматически генерирует `config.json` для Xray, следит за лимитами IP (анти-фрод) и отправляет статистику трафика на Master API.
* **Deploy Script (`cluster.sh`)**: Утилита для автоматической установки компонентов, добавления/удаления серверов по SSH и настройки инфраструктуры.

## Ключевые возможности

* **Протоколы**: VLESS (TCP/xHTTP) + XTLS-Reality, Shadowsocks-2022 (резервный), Classic TLS (для локального моста).
* **Балансировка**: RU-мосты автоматически распределяют трафик (LeastPing) между подключенными EU-нодами.
* **Управление через Telegram**: Встроенный бот для создания инвайтов, выдачи подписок и просмотра статистики.
* **Анти-фрод**: Автоматическая блокировка пользователей при превышении лимита активных устройств (парсинг логов через regex).
* **Обход блокировок**: Встроенный Cloudflare WARP на EU-нодах для скрытия IP нод.
* **CI/CD**: Автоматическая компиляция бинарных файлов (Master и Agent) через GitHub Actions при создании релиза.

## Требования к серверу (Мастер)

* Чистая ОС: Ubuntu 22.04+ / Debian 11+ (проверено на Debian 12)
* Привязанный поддомен (A-запись направлена на IP сервера).
* Telegram Bot Token (получить у @BotFather) и ваш Chat ID (получить у @getmyid_bot).

## Быстрый старт

1. Скачайте и запустите скрипт управления на главном сервере:
   ```bash
   wget -q https://raw.githubusercontent.com/sud0-i/KVN-XRAY-CLUSTER/main/cluster.sh -O cluster.sh
   chmod +x cluster.sh
   ./cluster.sh
   ```
2. В появившемся меню выберите 1. 🛠 Установить Master API и следуйте инструкциям.
3. После установки Мастера добавьте локальный мост или подключите удаленные серверы (EU-ноды) через соответствующие пункты меню.
